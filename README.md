# ![Rock Paper Scissors Game](image.png)Rock Paper Scissors Game

## 📌 Description

This is an interactive **Rock Paper Scissors** game built using:

- HTML
- CSS
- JavaScript

The player competes against the computer.  
The computer randomly selects a move, and the winner is determined based on the classic game rules.

---

## 🎮 How to Play

1. Open the `index.html` file in your browser.
2. Click one of the buttons:
   - Rock 🪨
   - Paper 📄
   - Scissors ✂️
3. The computer will randomly choose its move.
4. The result and score will be displayed on the screen.
5. Keep Playing to increase your score!



---

## 🛠 Technologies Used

- HTML5
- CSS3 (Flexbox, Background Styling)
- JavaScript (Game Logic, DOM Manipulation)

## 📜 Game Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choices result in a draw
